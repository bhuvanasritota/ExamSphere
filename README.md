# 🎓 ExamSphere

## Distributed Online Examination Platform

ExamSphere is a scalable online examination platform built using Java and Spring Boot. It supports secure authentication, live examinations, real-time monitoring, automatic evaluation, and analytics.

---

## 🚀 Tech Stack

- Java 17
- Spring Boot
- Spring Security
- PostgreSQL
- Maven
- WebSocket
- Redis
- RabbitMQ
- Docker
